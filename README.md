# Modern Calculator 🧮

This project is a **modern, responsive calculator** built using **HTML, CSS, and JavaScript**.  
It performs basic arithmetic operations with a clean UI and smooth user experience.

## 🚀 Features
- Responsive design for desktop and mobile
- Supports addition, subtraction, multiplication, and division
- Clear and reset functionality
- Keyboard input support
- Stylish UI with hover and click effects

## 🛠️ Technologies Used
- **HTML5** → Structure of the calculator
- **CSS3** → Styling, layout, and responsiveness
- **JavaScript (ES6)** → Logic for calculations and interactivity
